
## Video
We embedded videos in using two different methods: using the video tag and using iframe tag from Youtube and Vimeo.
We also add external buttons for

  * Play
  * Pause
  * Fast Forward
  * Rewind
  * Restart
  * A Fun button for panda sneezing.
